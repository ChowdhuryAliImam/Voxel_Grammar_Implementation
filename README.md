This is an computer implementation of 8 simple voxel rules. The first four rule is about adding masses, and the remaining four rulesa are about eroding mass. 
Then, I developed a random rule sequence and stored them in csv files.
To implement the code in your machine, you need to use Rhino 3D software as it used Rhinoscript syntax library. 
This code will read the rule sequences from the csv files, and generate forms following the rules. The generated designs will also saved in a targetted file. Change the file paths.
